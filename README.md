# Customer-Intelligence-Analytics-System
A comprehensive analytics platform that performs RFM (Recency, Frequency, Monetary) segmentation on customer transaction data, calculates customer lifetime value (CLV), and delivers actionable insights through interactive dashboards. The system will process 50,000+ customer transactions and provide segment-based treatment strategies.
